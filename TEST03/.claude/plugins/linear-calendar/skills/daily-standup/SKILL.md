# /daily-standup

오늘의 스탠드업 브리핑을 생성합니다.

## 실행 순서

1. `mcp__calendar__list_events` 로 오늘 일정 조회
2. `mcp__linear__list_issues` 로 전체 이슈 조회 (내 담당: assignee=mem-1)
3. `mcp__todo__list_todos` 로 미완료 할일 조회

## 출력 형식

아래 형식으로 브리핑을 출력하세요:

```
📅 오늘 일정 (MM월 DD일)
──────────────────────
⏰ HH:MM  일정 제목
⏰ HH:MM  일정 제목

🔵 진행중인 이슈
──────────────────────
• [식별자] 제목

⭕ 오늘 할일
──────────────────────
• 할일 제목 (마감: MM/DD)

⚠️  주의사항
──────────────────────
• 오늘 마감인 이슈 또는 일정 충돌이 있으면 여기에 표시
```

오늘 날짜 기준으로 필터링하고, 데이터가 없는 섹션은 "없음"으로 표시하세요.
