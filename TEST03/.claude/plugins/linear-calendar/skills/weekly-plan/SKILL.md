# /weekly-plan

이번 주 계획표를 자동으로 생성합니다.

## 실행 순서

1. `mcp__calendar__list_events` 로 이번 주 일정 조회
2. `mcp__linear__list_issues` 로 전체 이슈 조회
3. 이번 주 마감(due_date)인 이슈 필터링
4. 일정 + 이슈를 요일별로 정렬

## 출력 형식

```
📋 이번 주 계획 (MM/DD ~ MM/DD)
══════════════════════════════

월요일 (MM/DD)
  📅 일정: ...
  🎯 마감: [이슈식별자] 제목

화요일 (MM/DD)
  📅 일정: ...
  ...

──────────────────────
📊 이번 주 이슈 현황
  전체: N개 | 진행중: N개 | 완료: N개

💡 이번 주 집중 추천
  우선순위 높고 마감 임박한 이슈 TOP 3
```

이번 주 = 이번 주 월요일 ~ 일요일 기준으로 계산하세요.
