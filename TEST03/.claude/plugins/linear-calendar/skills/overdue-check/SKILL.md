# /overdue-check

기한이 지난 이슈를 찾아 정리합니다.

## 실행 순서

1. `mcp__linear__list_issues` 로 전체 이슈 조회
2. 오늘 날짜 기준으로 due_date가 지난 이슈 필터링 (status가 done/cancelled 제외)
3. 심각도별 분류:
   - 🔴 7일 이상 초과
   - 🟠 3~6일 초과
   - 🟡 1~2일 초과

## 출력 형식

```
⚠️  기한 초과 이슈 현황 (오늘: MM/DD)
══════════════════════════════

🔴 심각 (7일+ 초과)
  • [식별자] 제목 — DD일 초과 (담당: 이름)

🟠 경고 (3~6일 초과)
  • [식별자] 제목 — DD일 초과

🟡 주의 (1~2일 초과)
  • [식별자] 제목 — DD일 초과

총 N개 이슈 기한 초과
```

## 후속 처리

목록 출력 후 사용자에게 물어보세요:
- "마감일을 연장할까요?" → 연장할 이슈 선택 시 `mcp__linear__update_issue` 로 due_date 수정
- "상태를 취소로 변경할까요?" → `mcp__linear__update_issue` 로 status=cancelled 변경
