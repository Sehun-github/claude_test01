# /meeting-prep

다음 회의를 위한 자료를 자동으로 준비합니다.

## 실행 순서

1. `mcp__calendar__list_events` 로 오늘 + 내일 일정 조회
2. "회의" 또는 "미팅"이 포함된 이벤트 선택 (여러 개면 사용자가 선택)
3. `mcp__linear__list_issues` 로 진행중(in_progress) + 검토중(in_review) 이슈 조회
4. 회의 시간 전에 마감인 이슈 강조 표시

## 출력 형식

```
📋 회의 준비 자료
회의명: [일정 제목]
일시: MM/DD HH:MM
──────────────────────

📊 현재 진행 현황
  진행중 (N개):
  • [식별자] 제목 [담당자]

  검토 필요 (N개):
  • [식별자] 제목 [담당자]

  이번 주 완료 (N개):
  • [식별자] 제목

⚠️  이슈사항
  • 기한 초과 이슈
  • 담당자 없는 이슈
  • 블로킹 이슈 (있다면)

💬 회의 안건 초안
  1. 진행중 이슈 상태 공유
  2. 검토 필요 항목 논의
  3. 다음 액션 아이템 결정
```
